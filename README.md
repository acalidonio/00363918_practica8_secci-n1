¿Cuál es la diferencia entre autenticación y autorizacion?
La diferencia es que al autenticarse, se verifica la identidad de uno (iniciar sesión) y la autorización es verificar los permisos de uno (el middleware)

¿Cuál es la función del token JWT en la guía?
El JWT es el código que permite conectar correctamente la autenticación de cada usuario con su autorización.
